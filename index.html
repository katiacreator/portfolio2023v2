Studio Cyres is a Philadelphia based, front-end design agency led by Katia "Cyres" Gibson.

With Cy's background as a full-stack software engineer, sociologist + abstract photographer, Studio Cyres aims to create dynamic apps + visuals that are accessible + affordable to different communities.
